[js-bisect](http://aureooms.github.io/js-bisect)
==

Python's binary search library for JavaScript. Parent is
[js-algorithms](https://github.com/aureooms/js-algorithms).


```js
let {
	bisect_left ,
	bisect_right ,
	insort_left ,
	insort_right ,
} = bisect ;
```
